# 🎲 RPGManager

O **RPGManager** é um sistema de gerenciamento de RPG desenvolvido em Python, com foco em facilitar a organização de campanhas, personagens e sessões de jogo.

---

## 🚀 Funcionalidades

- Cadastro de personagens
- Gerenciamento de campanhas
- Controle de sessões de RPG
- Organização de jogadores
- Sistema simples e intuitivo

---

## 🛠️ Tecnologias utilizadas

- Python
- Django (ou Flask, se for o caso)
- SQLite (banco padrão)
- HTML / CSS (caso tenha frontend)
